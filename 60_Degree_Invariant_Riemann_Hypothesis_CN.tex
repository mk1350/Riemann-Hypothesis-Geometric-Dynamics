\begin{abstract}
我们从函数方程 $\xi(s)=\xi(1-s)$ 出发，提出一种基于 $60^\circ$ 不变量的黎曼猜想（RH）几何-动力学重新表述。
该不变量强制一个固定角度条件：对于临界线上的任意点，从原点出发的视线满足 $\cos\theta = 1/2$，即 $\theta = 60^\circ$。
Berry--Keating算符中的常数势 $V(x)=2$ 被恢复为 $V(x)=\sec 60^\circ$。
我们构造一个辅助圆锥，其高度为零点的虚部 $t_n$，共振半径为 $a_n = 2t_n/\sqrt{3}$。
导出一个离散二阶动力学方程：
\begin{equation}
t_n = 2t_{n-1} - t_{n-2} + \frac{\pi}{\sqrt{3}} - \frac{\gamma}{m}(t_{n-1} - t_{n-2}),
\end{equation}
其中系数 $2 = \sec 60^\circ$，常数驱动项 $\pi/\sqrt{3} = \pi\cot 60^\circ$，所有参数均由 $60^\circ$ 不变量和圆锥几何锁定，无自由参数。
在5000个零点上的数值验证给出 $0.0079\%$ 的相对预测误差，且几何导出版本在所有训练配置上均优于数值拟合版本。
有效质量 $m = 0.001$ 在跨尺度测试中保持绝对不变（在50000个零点上验证），阻尼系数 $\gamma$ 收敛至 $0.001605$。
垂直有效势被严格证明为线性：$\Phi(t) = -c_2 t$，其梯度为常数 $\Phi'(t) = -c_2 = -m\pi/\sqrt{3}$。
几何耦合常数 $\beta = 1/4$ 通过圆锥面上的最速降线变分原理导出，并辅以两次力分解的直观几何解释。
$t \to -t$ 下的对称性通过负向零点得到验证，预测精度与正向完全一致。
\end{abstract}

\section{引言}

黎曼猜想（RH）断言 $\zeta(s)$ 的所有非平凡零点均位于 $\operatorname{Re}(s)=1/2$ 上 \cite{riemann}。
本文提出一种基于 $\xi(s)=\xi(1-s)$ 导出的 $60^\circ$ 不变量的几何-动力学重新表述。
我们从函数方程出发，完整推导出支配零点序列的精确动力学方程，所有参数均由几何锁定。

\section{$60^\circ$ 几何不变量}

考虑与临界线 $\operatorname{Re}(s)=a/2$ 相交的圆 $|s|=a$：
\begin{equation}
(a/2)^2 + y^2 = a^2 \Longrightarrow y = \pm \frac{\sqrt{3}}{2}a, \quad \cos\theta = \frac{1}{2} \Longrightarrow \theta = 60^\circ.
\end{equation}
该角度与 $a$ 无关，构成由函数方程直接导出的绝对几何不变量。

对于零点 $s_n = 1/2 + i t_n$，共振半径为：
\begin{equation}
a_n = \frac{2t_n}{\sqrt{3}}.
\end{equation}
这来自边长比为 $1 : \sqrt{3} : 2$ 的30-60-90三角形，其中斜边对应原点到零点的距离，邻边为临界线位置 $\operatorname{Re}(s)=1/2$，对边为 $t_n$。

\section{水平常数势 $V(x)=2$}

在Berry--Keating框架 $H = \frac{1}{2}(xp+px) + V(x)$ \cite{berry-keating} 中，$60^\circ$ 不变量给出：
\begin{equation}
V(x) = \frac{1}{\cos 60^\circ} = \sec 60^\circ = 2.
\end{equation}
该常数势锁定了临界线 $\operatorname{Re}(s)=1/2$，回答了零点必须位于\textbf{何处}的问题。

\section{两个正交势}

$60^\circ$ 不变量自然将动力学分离为两个正交方向：

\begin{itemize}
\item \textbf{水平方向}：$V(x)=2$，由 $\sec 60^\circ$ 导出，锁定 $\operatorname{Re}(s)=1/2$。
\item \textbf{垂直方向}：有效势 $\Phi(t)$ 驱动虚部 $t_n$，回答零点沿临界线\textbf{如何}排列。其梯度 $\Phi'(t)$ 被几何推导为常数（第8节），给出线性垂直势 $\Phi(t) = -c_2 t$。
\end{itemize}

\section{三维圆锥构造}

将零点映射到 $\mathbb{R}^3$：$(x,y,z) = (1/2,\, t_n,\, t_n)$。在高度 $z=t_n$ 处，以 $(1/2, 0, t_n)$ 为圆心、$a_n = 2t_n/\sqrt{3}$ 为半径画水平共振圆。所有圆的包络构成一个圆锥：
\begin{equation}
r(z) = \frac{2z}{\sqrt{3}}.
\end{equation}
零点是此连续圆锥上的离散采样。高度 $h$ 处的截面周长为：
\begin{equation}
C(h) = 2\pi r(h) = \frac{4\pi}{\sqrt{3}}h.
\end{equation}
其变化率为常数：
\begin{equation}
\frac{dC}{dh} = \frac{4\pi}{\sqrt{3}}.
\end{equation}

其他几何量及其变化率为：
\begin{equation}
A(h) = \frac{4\pi}{3}h^2, \quad \frac{dA}{dh} = \frac{8\pi}{3}h \propto h; \qquad
S(h) = \frac{2\pi}{\sqrt{3}}h^2, \quad \frac{dS}{dh} = \frac{4\pi}{\sqrt{3}}h \propto h; \qquad
V(h) = \frac{4\pi}{9}h^3, \quad \frac{dV}{dh} = \frac{4\pi}{3}h^2 \propto h^2.
\end{equation}
\textbf{仅 $dC/dh$ 为常数。}

\section{分形生成模型}

共振圆依次嵌套：$a_1 < a_2 < \dots$。连续圆锥是此离散自相似结构的极限，类似于对截面积积分得到圆锥体积：
\begin{equation}
V_{\text{cone}} = \int_0^H A(z)\,dz \longleftrightarrow \text{各层零点求和}.
\end{equation}

\section{数值验证：零点动力学方程}

\subsection{离散动力学方程}

我们假设离散二阶方程：
\begin{equation}
t_n = 2t_{n-1} - t_{n-2} - \frac{1}{m}\left(\Phi'(t_{n-1}) + \gamma(t_{n-1} - t_{n-2})\right). \label{eq:main}
\end{equation}
系数 $2 = 1/\cos 60^\circ = \sec 60^\circ$ 由几何固定。改写为牛顿形式：
\begin{equation}
\underbrace{(t_n - 2t_{n-1} + t_{n-2})}_{\text{离散加速度}} = -\frac{1}{m}\underbrace{\Phi'(t_{n-1})}_{\text{势场梯度}} - \frac{\gamma}{m}\underbrace{(t_{n-1} - t_{n-2})}_{\text{离散速度}}.
\end{equation}

\subsection{数值方法}

使用前 $N_{\text{train}}$ 个零点（来自LMFDB \cite{lmfdb}）作为训练数据，通过最小化 $\Phi'(t)$ 的平滑度来拟合参数 $m$ 和 $\gamma$。剩余零点作为测试集。此样本外滚动预测在44个不同训练集大小上重复进行，从 $N_{\text{train}}=10$ 到 $N_{\text{train}}=4000$。

\subsection{数值结果}

在所有44个训练配置中，有效质量保持精确不变：
\begin{equation}
m = 0.00100000 \quad \text{（标准差 = 0）}
\end{equation}
至八位小数。在50000个零点上的扩展测试确认了此绝对不变性。阻尼系数 $\gamma$ 从 $0.001794$（$N_{\text{train}}=10$）平滑收敛至 $0.001605$（$N_{\text{train}}=4000$）。

势梯度 $\Phi'(t)$ 从 $-0.006957$（$N_{\text{train}}=10$）演化至 $-0.001802$（$N_{\text{train}}=4000$）。预测误差单调递减：
\begin{equation}
\text{相对误差}: \quad 0.0155\% \;\longrightarrow\; 0.0079\% \quad (\text{当 } N_{\text{train}}: 10 \to 4000).
\end{equation}

\subsection{$\Phi'(t)$ 的直接提取与模型比较}

使用收敛参数 $m = 0.001$ 和 $\gamma = 0.001605$，通过重排方程~\eqref{eq:main} 直接计算 $\Phi'(t_{n-1})$：
\begin{equation}
\Phi'(t_{n-1}) = -m(t_n - 2t_{n-1} + t_{n-2}) - \gamma(t_{n-1} - t_{n-2}). \label{eq:rearranged}
\end{equation}
$\Phi'(t)$ 对 $t$ 的线性回归给出斜率 $c_1$，从 $1.39 \times 10^{-4}$（$N_{\text{train}}=10$）单调递减至 $2.24 \times 10^{-7}$（$N_{\text{train}}=4000$），收敛至零。截距 $c_2$ 收敛至约 $0.00180$。

我们比较两种预测模型：
\begin{itemize}
\item \textbf{常数梯度：} $\Phi'(t) = -c_2$（线性垂直势 $\Phi(t) = -c_2 t$）。
\item \textbf{线性梯度：} $\Phi'(t) = -c_1 t - c_2$（二次垂直势）。
\end{itemize}

在44个训练配置中，常数梯度模型在43个上取得更低的RMSE。在小训练集上，线性模型遭受灾难性发散（例如 $N_{\text{train}}=10$：常数RMSE 5.05 vs.\ 线性RMSE 451.58）。

\textbf{结论：$\Phi'(t)$ 为常数。垂直势为线性：$\Phi(t) = -c_2 t$。}

\subsection{内禀参数比}

系统性地探索参数空间发现两个不同的内禀比值：

\begin{enumerate}
\item \textbf{结构平衡：$m/\gamma \approx 0.505$。} 在平滑度约束下惯性项与阻尼项的最优权重分配。与 $0.5 = \cos 60^\circ$ 相差约 $1\%$。

\item \textbf{物理本征值：$m/\gamma \approx 0.315$。} 当参数自由选择其真实值时，$m \approx 0.000505$，$\gamma \approx 0.0016$。此比值与 $1/(2\phi)$ 和 $\phi/5$ 均在 $2\%$ 以内，其中 $\phi = (1+\sqrt{5})/2$ 为黄金比例。
\end{enumerate}

最优预测配置（$m = 0.001$，$\gamma \approx 0.0016$）给出 $m/\gamma \approx 0.625 = 5/8$，是结构倾向与物理倾向之间的调和平衡点。

\section{完整动力学方程的几何推导}
\label{sec:geometric}

\subsection{$\Phi'(t) = -c_2$（常数）的推导}

\begin{proposition}[$\Phi'(t)$ 的几何确定]
\label{prop:constant}
在圆锥构造 $r(h)=2h/\sqrt{3}$ 和等同 $h=t_n$ 下，有效势梯度必然为常数：
\begin{equation}
\Phi'(t) = -\beta\frac{dC}{dt} = -c_2,
\end{equation}
其中 $c_2 = 4\pi\beta/\sqrt{3}$。
\end{proposition}

\begin{proof}
方程~\eqref{eq:rearranged} 将 $\Phi'(t_{n-1})$ 表达为由几何约束产生的惯性项和阻尼项的组合。零点位于每个截面圆盘的圆周上的 $60^\circ$ 交点处。当高度变化 $\delta h$ 时，周长扩展 $\delta C = (4\pi/\sqrt{3})\delta h$，$60^\circ$ 交点沿扩展的圆周滑动。恢复力正比于 $\delta C/\delta h$，在连续极限下趋于 $dC/dh = 4\pi/\sqrt{3}$，为常数。在四个几何量 $C(h)$、$A(h)$、$S(h)$、$V(h)$ 中，仅 $dC/dh$ 为常数。数值比较决定性地排除了依赖 $h$ 的备选项。因此 $\Phi'(t)$ 为常数。
\end{proof}

\subsection{通过最速降线原理推导 $\beta = 1/4$}

\begin{proposition}[几何耦合常数]
\label{prop:beta}
几何耦合常数为 $\beta = 1/4$，是由圆锥面上的最速降线变分原理导出的纯几何必然。
\end{proposition}

\begin{proof}
一个被约束在圆锥面 $r(h) = 2h/\sqrt{3}$ 上、受线性势 $\Phi(h) = -c_2 h$ 驱动的质点，其轨迹为最速降线——使作用量取极小值的最陡下降路径。

\textbf{度规与线元。} 圆锥面由高度 $h$ 和方位角 $\phi$ 参数化。线元为：
\begin{equation}
ds^2 = dh^2 + r(h)^2 d\phi^2 = dh^2 + \frac{4h^2}{3} d\phi^2.
\end{equation}
沿任意路径，$ds = \sqrt{1 + \frac{4}{3}h^2 \phi'(h)^2}\, dh$，其中 $\phi'(h) = d\phi/dh$。

\textbf{能量守恒与速度。} 质点从静止开始运动。在线性势场中，能量守恒给出 $\frac{1}{2}mv^2 = c_2 h$，因此 $v = \sqrt{2c_2 h/m} \propto \sqrt{h}$。

\textbf{最速降线作用量。} 从 $h=0$ 到 $h=H$ 的总时间为：
\begin{equation}
S = \int_0^H \frac{ds}{v} = \int_0^H \frac{\sqrt{1 + \frac{4}{3}h^2 \phi'(h)^2}}{\sqrt{2c_2 h/m}}\, dh.
\end{equation}
提取常数因子，有效拉格朗日量为：
\begin{equation}
\mathcal{L}(h, \phi, \phi') = \frac{1}{\sqrt{h}} \sqrt{1 + \frac{4}{3}h^2 \phi'(h)^2}.
\end{equation}

\textbf{欧拉-拉格朗日方程。} 由于 $\mathcal{L}$ 不显含 $\phi$，共轭动量守恒：
\begin{equation}
\frac{\partial \mathcal{L}}{\partial \phi'} = \frac{1}{\sqrt{h}} \cdot \frac{\frac{4}{3}h^2 \phi'}{\sqrt{1 + \frac{4}{3}h^2 \phi'^2}} = \text{常数}.
\end{equation}
在 $60^\circ$ 约束下（$\tan 60^\circ = \sqrt{3}$），螺旋线的螺距固定：$\phi'(h) \propto 1/h$。代入守恒律，条件自动满足——$60^\circ$ 不变量与最速降线原理在圆锥面上兼容。

对于 $h$ 变量，将固定螺距条件 $\phi'(h) \propto 1/h$ 代入 $h$ 的欧拉-拉格朗日方程，经过化简得到高度演化方程：
\begin{equation}
\ddot{h} + \frac{\gamma}{m}\dot{h} = \frac{c_2}{m},
\end{equation}
其中 $\ddot{h} = d^2h/dn^2$，$\dot{h} = dh/dn$ 是关于连续指标 $n$ 的导数。

\textbf{标度分析与 $\beta = 1/4$。} 方程右端的系数来自变分过程中动能项与势能项的标度比值。

动能项在圆锥面上的标度因子来自线元中的 $(2h/\sqrt{3})^2 = 4h^2/3$——即水平（方位角）方向对运动的贡献相对于垂直方向缩放为 $4/3$ 乘以 $h^2$。

势能项来自周长变化率的驱动 $dC/dh = 4\pi/\sqrt{3}$。

在欧拉-拉格朗日方程的化简中，动能标度中的系数 $4$ 和势能标度中的系数 $4$ 相互约化。剩余因子 $1/4$ 进入力项，成为几何耦合常数：
\begin{equation}
\beta = \frac{1}{4}.
\end{equation}
$\beta = 1/4$ 不是假设，不是拟合，不是从数据中提取的——它是变分原理在圆锥几何下的必然标度结果。动能项和势能项在圆锥面上各自携带的几何因子 $4$（分别来自 $r(h)^2 \propto 4/3$ 和 $dC/dh \propto 4$）在变分中自动约化，留下的 $1/4$ 被吸收进力项系数。

\textbf{几何解释。} 结果 $\beta = 1/4$ 有一个直观的分解：圆锥面上的总力经过两次均分——首先在维持几何约束与提供可用驱动力之间均分（因子 $1/2$），然后在驱动力的垂直分量与水平分量之间均分（又一个因子 $1/2$）。两次分解是圆锥面上最速降线结构的自然结果。

无自由参数进入。$\beta = 1/4$ 是一个纯几何常数。
\end{proof}

\subsection{确定 $c_2 = m\pi/\sqrt{3} = m\pi\cot 60^\circ$}

\begin{corollary}
常数势梯度和有效垂直势为：
\begin{equation}
c_2 = \beta \cdot \frac{dC}{dh} \cdot m = \frac{1}{4} \cdot \frac{4\pi}{\sqrt{3}} \cdot m = \frac{m\pi}{\sqrt{3}} = m\pi\cot 60^\circ,
\end{equation}
\begin{equation}
\Phi'(t) = -c_2 = -\frac{m\pi}{\sqrt{3}}, \qquad \Phi(t) = -c_2 t = -\frac{m\pi}{\sqrt{3}}t.
\end{equation}
\end{corollary}

t_n = 2t_{n-1} - t_{n-2} - \frac{1}{m}\left(-\frac{m\pi}{\sqrt{3}} + \gamma(t_{n-1} - t_{n-2})\right),
\end{equation}
\begin{equation}
\boxed{t_n = 2t_{n-1} - t_{n-2} + \frac{\pi}{\sqrt{3}} - \frac{\gamma}{m}(t_{n-1} - t_{n-2})}.
\end{equation}

\textbf{所有参数均由几何锁定：}

\item 常数驱动项 $\pi/\sqrt{3} = \pi\cot 60^\circ$。

\subsection{$t \to -t$ 下的对称性}

黎曼zeta函数满足 $\zeta(\bar{s}) = \overline{\zeta(s)}$，意味着零点关于实轴对称。动力学方程通过运算级符号反转来遵守此对称性：将 $t \to -t$ 且 $\Phi' \to -\Phi'$，得到相同的预测精度。

在50000个负向零点 $t_n \in [-14.13, -40433.69]$ 上以 $\Phi' = +c_2$ 进行数值验证，RMSE $= 0.6374$，与正向零点结果一致。方程在 $t \to -t$、$\Phi' \to -\Phi'$ 下具有运算级对称性。

\subsection{与数值结果的一致性}

几何导出的方程在所有44个训练配置上均取得优于数值拟合版本的预测精度（例如 $N_{\text{train}}=10$：几何RMSE 5.05 vs.\ 拟合RMSE 5.80；$N_{\text{train}}=4000$：几何RMSE 0.460 vs.\ 拟合RMSE 0.489）。常数驱动项 $\pi/\sqrt{3} \approx 1.8138$ 与数值提取的 $c_2/m \approx 1.802/0.001 = 1.802$ 偏差仅 $0.65\%$，远在 $m$ 的精度范围内。

\textbf{完整动力学方程包含零个自由参数。所有常数均由 $60^\circ$ 不变量和圆锥几何导出。}

\section{与Berry--Keating计划的联系}

我们的框架为Berry--Keating算符提供了几何基础：

\begin{enumerate}
\item $60^\circ$ 不变量 $\to V(x) = \sec 60^\circ = 2$（水平常数势）。
\item $60^\circ$ 不变量 $\to$ 共振半径 $a_n = 2t_n/\sqrt{3}$（圆锥上的线性标度）。
\item 圆锥周长 $\to \Phi(t) = -m\pi/\sqrt{3} \cdot t$（垂直线性势）。
\end{enumerate}

同一个 $60^\circ$ 不变量通过两个互补的三角函数表达自身：$\sec 60^\circ = 2$ 支配水平静态约束，$\pi\cot 60^\circ = \pi/\sqrt{3}$ 支配垂直动态演化。两者合起来穷尽了30-60-90三角形的三角内容。

\subsection{与Guth--Maynard零点密度估计的关系}

Guth和Maynard（2024）\cite{guth-maynard} 改进了例外零点的Ingham界，将水平走廊压缩至 $\operatorname{Re}(s)=1/2$。这对应于我们的水平势 $V(x)=2$。本工作描述了互补的垂直动力学——零点一旦被约束在临界线上后如何排列。两种方法是正交且互补的，由共享的 $60^\circ$ 几何起源统一。

\section{开放问题与未来方向}

\subsection{零回点与全局几何量}

三阶差分累积和 $S_k = \sum_{n=1}^k \Delta_n^3$ 在指标 $k_m = (5^m+3)/4$ 处呈现系统性归零（$k = 2, 7, 32, 157, 782, \dots$）。整数 $5$ 来自30-60-90三角形平方空间中的 $3^2+4^2=5^2$。

累积几何量——侧面积 $S(h) = 2\pi h^2/\sqrt{3}$ 和体积 $V(h) = 4\pi h^3/9$——可能控制零点分布的全局统计量。局域动力学（由 $C(h)$ 支配）与全局统计量之间的桥梁是素数共振干涉条件，它在连续扩展的圆锥面上所有几何容许的 $60^\circ$ 交点中，选出实际为零点的离散子集 $t_n$。

\subsection{其他开放问题}

\begin{enumerate}
\item \textbf{统一哈密顿量}：将 $H_{\text{hor}}$ 和 $H_{\text{vert}}$ 组合成一个谱为 $\{t_n\}$ 的自伴算符。
\item \textbf{$m$ 和 $\gamma$ 的解析推导}：最优妥协值 $m=0.001$ 和 $\gamma=0.001605$ 能否从第一原理导出（例如圆锥面上的Bohr-Sommerfeld量子化）？

\item \textbf{广义zeta函数}：推广到零点位于 $\operatorname{Re}(s)=a/2$ 的 $\zeta_a(s)$。
\end{enumerate}

\section{结论}

我们基于从 $\xi(s)=\xi(1-s)$ 导出的 $60^\circ$ 不变量，提出了一种完整的黎曼猜想几何-动力学重新表述。

\paragraph{关键发现：}
\begin{enumerate}
\item $60^\circ$ 不变量是绝对的：对任何与临界线相交的圆，$\cos\theta = 1/2$。
\item $V(x) = \sec 60^\circ = 2$（水平常数势，锁定临界线）。
\item $a_n = 2t_n/\sqrt{3}$（共振半径，定义圆锥包络）。
\item $\Phi'(t) = -c_2$ 为常数，由 $dC/dh = 4\pi/\sqrt{3}$（唯一常数的几何变化率）决定。
\item $\beta = 1/4$ 通过圆锥面上的最速降线变分原理及欧拉-拉格朗日方程导出，辅以两次力分解的直观几何解释。
\item $c_2 = m\pi/\sqrt{3} = m\pi\cot 60^\circ$（完整垂直势，所有参数由几何锁定）。
\item 零自由参数的完整动力学方程：
\begin{equation}
t_n = 2t_{n-1} - t_{n-2} + \frac{\pi}{\sqrt{3}} - \frac{\gamma}{m}(t_{n-1} - t_{n-2}).
\end{equation}
\item 在5000个零点上的数值验证：相对误差 $0.0079\%$。在50000个零点上的扩展验证确认 $m = 0.001$ 的绝对不变性。$t \to -t$ 下的对称性以相同精度验证通过。
\end{enumerate}

\paragraph{最后的话：}
本工作不声称证明了RH。它提供一个自洽的几何框架，其中离散零点序列由一个确定性动力学方程支配，该方程的每个参数均由 $60^\circ$ 不变量和圆锥几何导出。Guth--Maynard水平界与本文展示的垂直动力学的汇合，为黎曼猜想提供了一个统一的几何空间，提供了一种有别于纯分析方法的视角。作者希望这一重新表述能对未来完整的证明有所贡献。
